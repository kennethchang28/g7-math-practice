# 國中七年級下學期・第三次段考 數學練習系統

一個**離線可用、自動出題**的數學練習網頁。每按一次就產生 **20 題選擇題**，作答後**自動批改**並附上**詳細解題步驟**，可**無限次重新出題**。

> 🔗 **線上直接使用**：<https://kennethchang28.github.io/g7-math-practice/>
> （手機、平板、電腦的瀏覽器都可以直接打開使用，免安裝。）

---

## 📚 範圍與難度

對應七年級下學期第三次段考四大主題，每份試卷四主題**各 5 題、交錯排列**：

| 主題 | 涵蓋題型 |
|------|----------|
| **一元一次不等式** | 解不等式、去分母、整數解個數、最大/最小整數解、應用題（預算·平均門檻·方案比較·連續整數）、同解判別、數線表示、反向求係數 |
| **統計（平均數·中位數·眾數）** | 三種代表值計算、加權平均、逆求資料、次數分配表、合併平均（含反求人數）、改值求新平均、平均=中位反求、極端值與代表值觀念 |
| **幾何圖形** | 餘角補角、對頂角、鄰補角、平行線截角（含轉折角）、三角形內外角與等腰、多邊形內外角（含反向綜合）、角柱/角錐/曲面立體、尤拉公式、多面體綜合、正方體展開圖 |
| **線對稱與三視圖** | 對稱軸數、座標鏡射、格線對稱完成圖、立體堆疊的前/上/側視圖、數方塊 |

**難度定位**：挑戰級（多步驟、觀念陷阱、應用與綜合題），適合追求高分／頂尖的學生自我訓練。

## ✨ 特色

- **題目即時生成**：不是固定題庫，每次都是新題，可一直練。
- **答案保證正確**：所有題目的答案皆由程式「依建構方式」運算產生；並以獨立測試程式進行 **超過 54 萬次** 的交叉驗算（重算答案、檢查選項唯一性與語意），全數通過。
- **自動批改＋詳解**：交卷後立即顯示分數、各主題答對率，每題附上完整解題步驟與常見錯誤提醒。
- **純前端、可離線**：只有 HTML/JS，無需網路與安裝。

## 🚀 使用方式

**方法一（最簡單）**：直接打開上面的線上網址。

**方法二（下載到本機）**：下載本專案後，用瀏覽器打開 `index.html` 即可（需與 `generators.js` 放在同一資料夾）。

## 🧪 給想驗證／修改題目的人

```bash
# 需要 Node.js
node test_generators.js 8000     # 大量生成並交叉驗算所有題型（應為「失敗: 0」）
node server.js                   # 本機啟動預覽伺服器 http://localhost:4321
node dump_samples.js 6           # 匯出每種題型的範例到 samples.txt
```

- `index.html` — 介面（出題、作答、批改、詳解）
- `generators.js` — 題目生成引擎（瀏覽器與 Node 共用）
- `test_generators.js` — 獨立交叉驗算測試
- `server.js` / `dump_samples.js` — 本機預覽與範例匯出工具
